## Spring Boot

- 최소한의 설정만으로도 third-party library 사용 가능
- 바로 실행 가능
- 스프링 부트에 웹 어플리케이션 서버 내장되어 있음 -> 따로 설치 필요 없음
- component scan
  - 본 프로젝트에 사용된 각종 컴포넌트들을 읽어옴
  - 사용 용도에 따라 다양한 클래스들 존재
  - bean 형태로 등록
  - 클래스의 인스턴스 생성 대신 스프링 컨테이너에 의해 생성되고 관리되는 것을 IOC라고 함
