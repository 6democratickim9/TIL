### NAT

> 네트워크 주소 변환

- **Network Address Translation**

- IP패킷의 TCP/UDP 포트 숫자와 소스 및 목적지의 IP 주소 등을 재기록

  => 라우터를 통해 네트워크 트래픽을 주고 받는 기술

- 네트워크의 성능에 영향

- #### NAT 사용 이유

  - 여러 대의 호스트가 하나의 공인IP 사용

    => 인터넷에 접속하기 위한 경우가 대부분

    -> 인터넷 공유기를 달아서 여러 PC 를 연결하여 사용 가능하게 만들어줌
